# DataTableSummary

This component renders a summary for McGonnagall question cards. It's primarily used to represent user selections in React Table form.

## Importing ðŸ“¦

You can use this component using one of the following import patterns.

```javascript
import DataTableSummary from '@unitedincome/components/dist/DataTableSummary';
```

```javascript
import {DataTableSummary} from '@unitedincome/components';
```
